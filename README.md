Learn how to build the famous eyes that follow the mouse cursor. 

![Image](thumb.png)

It is an awesome effect that can be implemented easily in CodeGuppy with the help of a little bit of trigonometry.
 
## Source code 
The JavaScript source code of this program can be found [here](sketches/program.js). To run it, you need to import it in the [codeguppy.com](https://codeguppy.com) online editor. Alternatively, you can use the source code in your p5.js sketches (advanced use). 
## Online version 
To see the code running, check the online playground at [https://codeguppy.com/code.html?t=eyes_mouse](https://codeguppy.com/code.html?t=eyes_mouse)
